# Asphalt Emissions Study (2018)

This project analyzes asphalt-related emissions and their environmental impact, focusing on anthropogenic secondary organic aerosol (SOA) formation and ozone production. The dataset includes emissions inventories from 2018 at the state and county level, providing a detailed spatial analysis of asphalt-related pollution.
